# Color picker
A program that helps with choosing colors and working with their codes.
Written in C++ using Qt 5 framework.

# Features
- Color spaces: RGB, HSV, HSL, CMYK.
- Qt standard color picker.
- Decimal/hexadecimal color codes.
- HTML color list.
